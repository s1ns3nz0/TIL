# MITRE D3FEND
## 1. What is the 'MITRE D3FEND'
- A knowledge graph of cybersecurity countermeasures
- https://d3fend.mitre.org/about/
## 2. Characteristics
- D3FENDER: Detection, Denial, Disruption Framework Empowering Network Defense
- 5 Techniques: Harden > Detect > Isolate > Deceive > Evict
- Enables building an attack and defense matrix by integrating with ATT&CK matrix based on artifacts
- Enables mapping attacker techniques to defense methods through scenario-based analysis
- Enables visualizing the entire flow using CAD
## 3. Learning Content
- How to draw diagrams using CAD examples (https://d3fend.mitre.org/cad/?load=../resources/cad/domain/ot/purdue-simple.json&render_meta=true)
- How to map artifacts to the ATT&CK matrix
